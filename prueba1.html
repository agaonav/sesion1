SUMILLA

La unidad didáctica Desarrollo Web corresponde al programa de Desarrollador Web Front-End, tiene carácter teórico-práctico. A través de ella se busca que el estudiante sea capaz de crear interfaces de usuario, desarrolladas con tecnología web basada en HTML y CSS, utilizando Bootstrap para gestionar proyectos web.

El desarrollo de la unidad didáctica es fundamental porque permitirá al estudiante participar de proyectos con manejo de control de versiones en un entorno Cloud Computing, implementar aplicaciones web estáticas y utilizar herramientas digitales para gestionar recursos y requerimientos, de acuerdo con la demanda actual de cualquier tipo empresa.
